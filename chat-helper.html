  1. Enter your JWT token above
  2. Click "Get All Users" to see available users (SuperAdmin, Admin, Agent)
  3. Copy another user's ID (can be any role)
  4. Paste it in "Create Chat" section and click "Create Direct Chat"
  5. Copy the chat ID from the result
  6. Use that chat ID in the chat-test-client.html

Note: All authenticated users (SuperAdmin, Admin, and Agent) can now participate in chats!
